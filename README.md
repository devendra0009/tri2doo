# Tri2Do
#### It is ONE-STOP Web-App for all trending DSA Sheets. 

## Demo
#### https://tri2do.vercel.app/

## Installation Guide
### clone the project
```
  git clone git@github.com:Ankur-Ydv/Tri2Do.git
```
### go to the project directory
```
  cd Tri2Do
```
### Install dependencies
```
  npm install
```
### start the app
```
  npm run dev
```
